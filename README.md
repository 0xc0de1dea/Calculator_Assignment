# 계산기 사용법

============= 계산기 프로그램입니다. =============
- 1. 간단한 계산하기
- 2. 계산식을 이용한 계산하기
- 3. 임의의 값보다 큰 계산식을 히스토리에서 찾기
- 4. 계산기 종료

와 같은 기능이 있습니다.

1번을 선택시 하나의 연산자와 두개의 피연산자를 입력받아 계산할 수 있습니다.
ex) 1 , + , 2

2번을 선택시 한줄의 계산식을 입력받아 한번에 여러개의 수식을 계산할 수 있습니다.
ex) 2 + 3 * 4 + ( 5 - 6 ) / 7

3번을 선택시 현재 입력받은 값보다 큰 값을 이전 히스토리에서 찾아 출력해 줍니다.

4번을 선택시 계산기가 종료됩니다.
